## 🏗️ Arquitectura

### Servicios Principales

1. **Integration System** (8000) - API Gateway
2. **Content Redundancy Detector** (8001)
3. **BUL** (8002) - Business Unlimited
4. **Gamma App** (8003)
5. **Business Agents** (8004)
6. **Export IA** (8005)

### Infraestructura
- PostgreSQL (Base de datos)
- Redis (Cache)
- Nginx (Load Balancer)
- Prometheus + Grafana (Monitoreo)
- ELK Stack (Logs)
