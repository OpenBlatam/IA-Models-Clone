```bash

```

```bash
# Instalar Redis
sudo apt-get install redis-server

# Iniciar Redis
sudo systemctl start redis-server
```

### **3. Configuración de Variables de Entorno**
```bash
export REDIS_URL="redis://localhost:6379"
export LOG_LEVEL="INFO"
export ENVIRONMENT="development"
```
