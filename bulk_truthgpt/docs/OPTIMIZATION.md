## ‚ö° **5. Lazy Loading System**

- **M√∫ltiples estrategias** de carga (On-demand, Preload, Cached, Lazy)
- **Prioridades** de carga (Low, Normal, High, Critical)
- **TTL** y expiraci√≥n autom√°tica

### **Estrategias de Carga:**
```python

```

### **Uso:**
```python

    key="expensive_data",
    loader_func=lambda: expensive_computation(),
    strategy=LoadingStrategy.ON_DEMAND,
    priority=LoadingPriority.HIGH,
    ttl=3600
)

# Obtener valor

# Decoradores para propiedades y m√©todos

    @lazy_method(ttl=1800)

        return expensive_computation(param)
```
